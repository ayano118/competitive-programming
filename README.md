# AtCoder

AtCoderの精進記録です。解いた問題のコードを Python でまとめています。

## 🔗 プロフィール

- AtCoder: https://atcoder.jp/users/NoAAA

## 📁 構成

コンテストごとにフォルダを分けて管理しています。

```
abc300/
  a.py
  b.py
  c.py
arc180/
  a.py
```

- `abc` … AtCoder Beginner Contest
- `arc` … AtCoder Regular Contest
- ファイル名は問題（a, b, c, ...）に対応
